Site Privacy Policy

Last updated: March 28, 2026

1. Introduction

This site privacy policy describes how the Clipor website (hereinafter referred to as "this Site") handles visitor information. The operator of this Site (hereinafter referred to as "we") is committed to protecting visitor privacy.

For the privacy policy of the Clipor application itself, please refer to the App Privacy Policy.

2. Site Hosting

This Site is hosted on GitHub Pages, a service provided by GitHub, Inc. GitHub may collect technical information such as IP addresses and browser information in accordance with their own privacy policy.

  • GitHub Privacy Statement: https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement

3. Information We Collect

This Site does not independently collect, store, or process any personal information from visitors. Specifically:

  • No cookies are set by this Site
  • No analytics or tracking tools are used
  • No user registration or login functionality exists
  • No forms that collect personal information are provided

4. Local Storage

This Site uses the browser's localStorage feature solely to save your language preference (English/Japanese). This data is stored only in your browser and is not transmitted to any server.

  • Key: clipor-lang
  • Value: 'en' or 'ja'
  • You can clear this data at any time through your browser settings

5. External Links

This Site contains links to external websites such as GitHub and Google Fonts. These external sites have their own privacy policies, and we are not responsible for their content or privacy practices.

6. Third-Party Services

This Site uses the following third-party services:

  • Google Fonts - for web font delivery. Google may collect usage data in accordance with their privacy policy (https://policies.google.com/privacy).
  • GitHub Pages - for site hosting (see Section 2)

7. Children's Privacy

This Site is not targeted at children under the age of 13. Since this Site does not collect any personal information, it does not knowingly collect data from children.

8. Changes to This Policy

We may update this site privacy policy from time to time. Any changes will be posted on this page with an updated revision date. Continued use of this Site after changes constitutes acceptance of the updated policy.

9. Contact

If you have any questions or concerns about this site privacy policy, please contact us through the following:
