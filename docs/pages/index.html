Clipor is a blazing-fast, lightweight clipboard manager for Windows. Built with Rust and Tauri for native performance with minimal resource usage.

Everything you need

A powerful clipboard manager that stays out of your way until you need it.

Clipboard History

Automatically records everything you copy. Search, filter, and browse up to 1,000 entries with full-text search.

Pin Favorites

Pin frequently used entries to keep them at the top. Pinned items are protected from automatic cleanup.

Smart Templates

Save reusable text snippets with dynamic variables like {{date}}, {{time}}, and {{clipboard}}. Organize into groups.

Password Protection

Encrypt your clipboard history with AES-256-GCM. PBKDF2 key derivation ensures your data stays secure.

Global Hotkey

Press Ctrl+Alt+Z to instantly access your clipboard history from any application. Fully customizable.

Text Transform

Convert between full-width and half-width characters, trim whitespace, and normalize text on the fly.

Image Support

Copies images too. View thumbnails in your clipboard history and paste them whenever you need.

Hover Preview

Hover over any entry to see a full preview tooltip. No need to click or open a separate window.

Built with modern tech

Combining the best of Rust's performance with React's developer experience.

Backend
Rust + Tauri 2

Native Windows performance with a tiny footprint. No Electron, no bloat.

Frontend
React + TypeScript

Type-safe UI with React 19 and Vite for instant hot module replacement.

Database
SQLite

Embedded database with rusqlite. Zero configuration, zero external dependencies.

Security
AES-256-GCM

Military-grade encryption with PBKDF2 key derivation. Your data stays local and secure.

Get started in seconds

Clipor runs quietly in your system tray, ready when you need it.

1

Install

Download the MSI or NSIS installer and run it. Clipor launches automatically.

2

Copy anything

Use your computer as normal. Clipor silently records your clipboard in the background.

3

Recall instantly

Press Ctrl+Alt+Z to open the popup. Select any entry to paste it.
